### O BlackHole é um aplicativo criado em Python que utiliza a biblioteca pytubefix. Ele permite o download de Vídeos e Músicas do Youtube em instantes, sem a presença de anúncios incômodos.

# Funcionalidades 🛠
* Compatibilidade com Windos no download .exe
* Compatibilidade para demais plataformas utilizando o arquivo .py
* Download de vídeos MP4
* Download de audios M4A
* Download Ilimitado
* Sem anúncios
* Interface Extremamente leve rodando direto do terminal
* Suporte ao Youtube Music
